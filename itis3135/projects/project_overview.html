Project Overview

Project Name

D&A Detailing

Project Overview

  • Purpose: A professional website for D&A Detailing, a local car detailing business that showcases services, builds credibility, and allows customers to schedule appointments or request quotes.
  • Intended Users: Vehicle owners in the local area looking for interior/exterior car detailing, including students, staff, and residents near UNC Charlotte.
  • Overview of Content: The website will include pages such as Home, About Us, Services, Gallery, and Contact. Users can learn about the business, view service packages, see before-and-after photos, and submit a form to request services or ask questions.

Site Map

Page Design

  • Home
    Purpose: Introduce D&A Detailing, highlight services, and provide quick access to contact info.
    Audience: All visitors and potential clients.
    Content: Welcome message, high-quality banner image, brief list of services, call-to-action buttons.
    No form fields or validation.

  • About Us
    Purpose: Share background on the founders, experience, and mission of D&A Detailing.
    Audience: Visitors wanting to learn more about the business.
    Content: Bio, photo of the team, and company values.
    No interactivity.

  • Services
    Purpose: Outline detailing packages and pricing.
    Audience: Car owners looking for detailing services.
    Content: Accordion or tabs with service descriptions (Interior, Exterior, Full Detail).
    jQuery UI accordion used for interaction.

  • Gallery
    Purpose: Show before-and-after photos of detailed cars.
    Audience: Anyone interested in proof of work.
    Content: Thumbnails that open into larger views in a lightbox gallery.
    jQuery lightbox plugin used.

  • Contact
    Purpose: Allow visitors to request quotes or ask questions.
    Audience: Potential clients.
    Content: Contact form with name, email, phone, and message.
    JavaScript validation for required fields and proper formatting.

Dynamic Functionality

  • Form Validation: JavaScript will validate the contact form to ensure all fields are filled out and formatted correctly (especially email and phone).
  • Interactive Gallery: The Gallery page will use a jQuery Lightbox plugin to display images in a clean, modern popup view.
  • Accordion for Services: On the Services page, a jQuery UI accordion will let users toggle through different detailing packages.
  • Examples:
    • jQuery UI Accordion
    • Lightbox2 Gallery

Notes: This website will use a responsive layout, include consistent headers and footers with navigation and validation links, and validate cleanly through the W3C tools. AJAX may be used in the future to dynamically load testimonial or pricing content from a static JSON file.
